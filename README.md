# 🧊 G.AI - Gestão Auto Inteligente

> **A plataforma definitiva de inteligência para o mercado de gelo.**

O **G.AI** é uma solução de gestão (ERP/SaaS) desenvolvida especificamente para modernizar fábricas e distribuidoras de gelo. Com uma interface premium, mobile-first e funcionalidades focadas na agilidade do dia a dia, o sistema centraliza produção, PDV, estoque e financeiro em uma única plataforma inteligente.

---

## 🚀 Diferenciais do G.AI

- **Inteligência Artificial Nativa**: Assistente integrado para análise de lucros e sugestões estratégicas.
- **Mobile-First**: Projetado para ser usado em tablets e celulares por motoristas e vendedores.
- **Multi-Depósito**: Controle total de estoque em múltiplas unidades e filiais.
- **PDV de Atacado & Varejo**: Interfaces dedicadas para diferentes canais de venda.

---

## 🛠️ Funcionalidades Principais

### 1. 📊 Dashboard BI
- Visão geral do faturamento e indicadores de saúde do negócio.
- Alertas preditivos de estoque.

### 2. 🛒 Ponto de Venda (PDV)
- **Varejo**: Rapidez para balcão e delivery.
- **Atacado**: Gestão de pedidos grandes, comissões de vendedores e recibos térmicos.

### 3. 📦 Produção e Estoque
- Controle de lotes de Gelo Cubo e Sabor.
- Gestão de perdas e transferências entre unidades.

### 4. 💰 Financeiro & CRM
- Fluxo de Caixa, DRE e controle de despesas.
- Cadastro de clientes com segmentação automática e histórico de compras.

---

## 💻 Stack Tecnológica

- **Frontend**: React + TypeScript + Vite
- **Estilização**: Vanilla CSS + Tailwind (Navy/Teal Design System)
- **Backend**: Supabase (PostgreSQL, RLS, Edge Functions)
- **IA**: Google Gemini Pro (Análise Estratégica)

---

## 🚀 Como Iniciar

1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Configure o `.env` com as chaves do Supabase e Gemini.
4. Rode localmente: `npm run dev`.
